# About SynDORBench

We introduce SynDORBench, a framework and synthetic dataset designed to evlauate the effective operating range of Large Vision Language Models(LVLMs) on Detection and Observation Ranges for Benchmarking. The ranges are calibrated to human visual thresholds, facilitating a comparison of human visual capability to LVLM perception.
