## for example evaluation with models

1. create a conda env
```
conda create -n syndorbench_env python=3.12.2
```
2. activate the environment
```
conda activate syndorbench_env
```
3. install the packages
```
pip install -r requirements.txt
```
If you intend to run models sourced from external repositories, please set up the environment according to the instructions provided in those repositories.
